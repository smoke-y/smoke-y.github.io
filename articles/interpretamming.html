Hamming code is a way of detecting errors in the received encoded data. Interpretation 1 SUMMARY: adding parity bits to the original data allows you to drill down on the location of the error in a byte stream. Interpretation 2 SUMMARY: transform data to a higher-order representation so that the distance between any 2 valid points increases. If we get a vector that is not a valid point, we change the value to the valid point that the vector leans toward(Valid points create a lattice in the higher dimension). Interpretation 1 comes from digital system design, whereas interpretation 2 comes from discrete mathematics.
